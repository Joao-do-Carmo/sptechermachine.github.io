SPTecher's Machine

Converter Calculator Info

Descrição

Este projeto foi desenvolvido como parte da disciplina Arquitetura Computacional. O SPTecher's Machine é uma ferramenta que permite conversão entre diferentes bases numéricas (binária, octal, decimal e hexadecimal), além de oferecer uma calculadora para realizar operações aritméticas em diferentes bases numéricas. O projeto foi desenvolvido para facilitar a compreensão e manipulação de números em vários sistemas numéricos.

Funcionalidades

Conversão entre bases binária, octal, decimal e hexadecimal.
Calculadora que permite realizar operações em diferentes bases numéricas (soma, subtração, multiplicação e divisão).

Tecnologias Utilizadas

HTML: Estruturação da página.
CSS: Estilização da interface.
JavaScript: Lógica para conversão de bases e cálculo de operações numéricas.
